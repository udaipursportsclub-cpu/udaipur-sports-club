# Udaipur Sports Club

A web platform for hosting and managing sports events in Udaipur, India.

---

## Getting Started

Follow these steps to run the project on your computer:

**1. Install everything the project needs**
```bash
npm install
```

**2. Add your secret settings**
Open the `.env.local` file and fill in your real Supabase values.
(The file already exists — just replace the placeholder text.)

**3. Start the website on your computer**
```bash
npm run dev
```

**4. Open it in your browser**
Go to: [http://localhost:3000](http://localhost:3000)

You should see the Udaipur Sports Club welcome screen.

---

## Tech Stack (what this is built with)

| Tool | What it does |
|---|---|
| Next.js 14 | The main framework — builds and runs the website |
| TypeScript | Makes the code safer by catching mistakes early |
| Tailwind CSS | Makes the website look nice without writing much CSS |
| Supabase | Handles the database and user login |
